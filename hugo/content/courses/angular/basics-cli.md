The [Angular CLI](https://angular.io/cli) provides a powerful set of tools for generating code from the command line. 

## Getting Started

{{< file "terminal" "command line" >}}
{{< highlight terminal >}}
npm install -g @angular/cli

ng --version
ng help

ng generate component --help

ng new my-app
code my-app
{{< /highlight >}}
